# Okinawa Fitness

Okinawa Fitness — marketing site built with Vite + React + TypeScript.

Local development

```bash
npm install
npm run dev
```

Prepare and push to GitHub

1. Create a new repository on GitHub (via the website).
2. Add remote and push:

```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin git@github.com:USERNAME/REPO_NAME.git
git branch -M main
git push -u origin main
```

If you prefer, install GitHub CLI (`gh`) and run `gh repo create` to automate remote creation.
